Feature Description

Better Admin Page

Key statistics that we would like to see/access on the Admin Page so that we have a better understanding of activity by Groups and individual Users.   

Group View Statistics 

  • Number of projects
  • Number of users
  • Number of saves
  • Number of builds
  • Type of build(s)
  • Were changes made to the CSS file(s)
  • First and Last commit

Individual User View Statistics 

  • Number of invites sent & accepted
  • Number of projects created 
  • Number of saves
  • Number and type of build(s) by user
  • First and Last commit
