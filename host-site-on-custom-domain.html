Feature Description

In the current version of Atlas, we can generate an HTML version of a site that the client can then download and put on their own hosting service.  As part of exploring how we might publish our own sites, we've created service called the Atlas Web Publisher to post HTML content to an S# bucket so that it can be viewed.  You put in your atlas API key and the project name, and the site is published to "sites.oreilly.com/your-atlas-account-name/your-project-name/".
